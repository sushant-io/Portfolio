# Sushant's Portfolio

This portfolio highlights my personal details, previous projects, the tech stack I can handle, a contact form through which you can reach me, and my CV along with working links to my projects.

## Live Demo
[View Portfolio](https://sushant-io.github.io/Portfolio/)

## Features / Sections
- **Navbar & Hero Section** – My photo, short details, and social buttons (LinkedIn, GitHub, Instagram)  
- **Tech Stack** – Technologies I can handle with icons and names  
- **Projects** – 5 projects with working links  
- **Contact** – Contact form, email, phone, CV download link, and social media buttons
